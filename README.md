# Credituz Comissões

Apresentação interativa do Programa de Comissões Credituz por plano, publicada em https://comissoes.credituz.ai via GitHub Pages.

- Planos: Free 10%, Starter 13%, Growth 15%, Business 25%, Enterprise 40 a 50% (exibido como 40%)
- Base privadas: 0,75% sobre o valor financiado
- Base públicas (Caixa e BB): R$ 500 por pasta enviada
- Simulador exibe a taxa efetiva do parceiro sobre o valor total financiado

## Clarity

O tag do Microsoft Clarity está comentado no `index.html`. Para ativar, substituir `CLARITY_ID` pelo ID do projeto e remover os comentários.
